Projeto: Exo.WebApi
Aluno: Roberto Micaías
Descrição: API REST para gerenciamento de projetos, com integração ao banco SQL Server.
Endpoints disponíveis:
- GET /api/projetos
- GET /api/projetos/{id}
- POST /api/projetos
- PUT /api/projetos/{id}
- DELETE /api/projetos/{id}

Banco de dados: ExoApi (SQL Server)
Tabela: Projetos (Id, NomeDoProjeto, Area, Status)
