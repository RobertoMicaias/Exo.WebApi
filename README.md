# Exo.WebApi

API REST desenvolvida em .NET 8.0 para gerenciamento de projetos.  
Integração com banco de dados SQL Server.

## Endpoints:
- GET /api/projetos
- GET /api/projetos/{id}
- POST /api/projetos
- PUT /api/projetos/{id}
- DELETE /api/projetos/{id}

## Tecnologias:
- C#
- .NET 8.0
- Entity Framework Core
- SQL Server
- Swagger UI
